How It Works

Discover the AI and data pipeline powering optimal renewable energy placement.

Step-by-Step Process

  1. Data Collection: We use Nasa collected weather data, then isolate it to a specific area. We then compare it to publically availible records on land availibility. Giving us a substantial ammount of raw data.
  2. Preprocessing: We then combine this raw data with user inputs.
  3. AI Model: Then we pass this data bomb onto a AI model, which then analysises the data and passes interpretation to the user through the chat feature
  4. Location Ranking: The chat bot will rank the results and then place display it to the user in simplified words, which anyone can understand (Ie no need for a masters in statistics to understand).
  5. Visualization: Finally we then display the results on a interactive map.

System Architecture

We use a chat bot trained through gemini Ai to allow for the chat feature, a _____ library for the interactive map, and nasa statistics for the raw data.
