How It Works

Discover the AI and data pipeline powering optimal renewable energy placement.

Step-by-Step Process

  1. Data Collection: Using a specified longitude and latitude value, Aneious collects relevent weather info from Nasa's API on climate data.
  2. Preprocessing: We then combine this raw data with user inputs, to prevent the temperature or wind speed from causing to much variance outside of the cells operational limits.
  3. Gemini AI studio Model:Then we pass this data bomb onto a AI model, which then analysises the data and passes interpretation to the user through the chat feature
  4. Cesium ion Interactive map: This same data is simaltaniously being sent to a cesium ion interactive map. Allowing for the user to both compare the location specified by the AI, and check its accuracy
  5. User conclusion: given this data the user can then enter variables until they find a spot which will work with their projects development

System Architecture

We use a chat bot trained through gemini Ai to allow for the chat feature, a Cesium Ion library for the interactive map, and NASA statistics for the raw data.
